## 3. 데이터 처리 파이프라인

### 3.1 처리 흐름

```
Excel 파일 (연도별)
    │
    ▼
[전처리] 헤더 행 위치 탐지 → skiprows 적용
    │
    ▼
[검증] 조사기준일 앞 4자리로 실제 연도 확인
    │
    ▼
[필터링] 폐(원)교 · 휴(원)교 행 제거
    │
    ▼
[보정] D열 교육지원청명 — 고등학교급 광역명 → 실제 지원청명으로 매핑
    │
    ▼
[분류] 학교급 10종 → 6개 표준 분류로 통합
    │
    ▼
[집계] 시도 × 학교급_분류 기준 학생수·학교수 합산
    │
    ▼
JSON 저장 (edu_data.json)
    │
    ▼
대시보드 HTML에 인라인 삽입
```

### 3.2 핵심 처리 로직

#### 이슈 1: 상단 설명문 (헤더 행 위치)

2009·2010·2012·2013·2014년은 상단 1~11행에 설명 텍스트가 있어 실제 헤더가 12행에 위치한다.

```python
# skiprows 값으로 처리
skiprows_map = {
    2009: 11, 2010: 11, 2012: 11, 2013: 11, 2014: 11,
    # 나머지 연도: 0
}
df = pd.read_excel(fname, skiprows=skiprows_map.get(year, 0))
```

#### 이슈 2: 학교수 집계 방식 분기

2021년 이전에는 `학교수` 컬럼이 없고 행 1개 = 학교 1개 구조다. 폐(원)교·휴(원)교를 먼저 제거한 후 학교명 기준으로 카운트한다.

```python
# 폐·휴교 제거 — 전 연도 공통 패턴: '폐(원)교', '휴(원)교'
df = df[~df['상태'].astype(str).str.contains('폐|휴', na=False)]

# 학교수 집계 분기
if '학교수' in df.columns:          # 2021년~
    agg_col, agg_func = '학교수', 'sum'
else:                                # 2008~2020년
    agg_col, agg_func = '학교명', 'count'

agg = df.groupby(['시도','학교급_분류']).agg(
    schools=(agg_col, agg_func),
    students=('학생수_총계_계', 'sum'),
    male=('학생수_총계_남', 'sum'),
    female=('학생수_총계_여', 'sum'),
).reset_index()
```

#### 이슈 5: 연도 확인 (시트명 오기재 대응)

2019년 파일은 시트명이 `2021년`으로 잘못 기재되어 있다. 시트명 대신 조사기준일로 판별한다.

```python
actual_year = int(str(df.iloc[0]['조사기준일'])[:4])
```

#### D열 교육지원청 보정 (전 연도 공통)

고등학교·고등기술학교·방송통신고등학교의 D열은 `서울시`, `경기`, `부산시` 등 광역명으로 표기된다. 초·중학교 데이터 기준으로 `(시도, 행정구) → 교육지원청` 매핑 테이블을 생성하여 역매핑한다.

```python
# 매핑 테이블 생성 (초·중학교 기준, 광역명 제외)
base = df[~df['학교급'].isin(['고등학교','고등기술학교','방송통신고등학교'])]
real = base[~base['교육지원청_원본'].str.contains(
    r'시$|도$|특별자치시$|특별자치도$', na=False)]
mapping = real.groupby(['시도','행정구'])['교육지원청_원본'].agg('mode').to_dict()

# 고등학교 보정 적용
def fix_edu(row):
    if row['학교급'] in ['고등학교','고등기술학교','방송통신고등학교']:
        return mapping.get((row['시도'], row['행정구']), row['교육지원청_원본'])
    ...
```

## 5. 구조 변화 이슈 목록

### 이슈 1: 상단 설명문 (헤더 행 위치)
- **대상**: 2009·2010·2012·2013·2014년 (5개 연도)

- **처리**: `skiprows=11` 적용

### 이슈 2: 학교수 집계 방식 전환
- **전환 시점**: 2020년 → 2021년
- **2008~2020년**: 행 1개 = 학교 1개, `학교수` 컬럼 없음 → `학교명` 카운트
- **2021년~**: `학교수` 컬럼 합산
- **공통 적용**: `폐(원)교`, `휴(원)교` 상태 행 전 연도 제거

### 이슈 3: 컬럼 수 4단계 증가
- **116개** (2008~2011): 기준
- **122개** (2012~2014): 수석교사 컬럼, 전체직원 컬럼 추가

### 8.1 체크리스트

```python
# 1. 첫 번째 시트가 데이터 시트인지 확인

# 3. 실제 연도는 시트명이 아닌 조사기준일로 확인
actual_year = int(str(df.iloc[0]['조사기준일'])[:4])

# 4. 컬럼 수 확인 (예상 범위: 116/122/125/147/148/149)
assert len(df.columns) in [116, 122, 125, 147, 148, 149, 150]

# 5. 상태 컬럼 값 확인 (폐·휴교 패턴 변경 여부)
print(df['상태'].value_counts())

# 6. 학교수 컬럼 유무 확인
has_school_col = '학교수' in df.columns
```

### 8.2 처리 스크립트 실행 순서


## 9. 데이터 처리 Python 함수 (표준)

```python
import pandas as pd, json, re

SIDO_ORDER = ['서울','부산','대구','인천','광주','대전','울산','세종','경기',
              '강원','충북','충남','전북','전남','경북','경남','제주']

LEVEL_MAP = {
    '유치원':'유치원', '초등학교':'초등학교',
    '중학교':'중학교', '방송통신중학교':'중학교',
    '고등학교':'고등학교', '고등기술학교':'고등학교', '방송통신고등학교':'고등학교',
    '특수학교':'특수학교', '각종학교':'기타', '고등공민학교':'기타'
}

def process_year(fname, skiprows=0):
    df = pd.read_excel(fname, skiprows=skiprows)

    # 연도 확인
    actual_year = int(str(df.iloc[0]['조사기준일'])[:4])

    # 폐·휴교 제거
    df = df[~df['상태'].astype(str).str.contains('폐|휴', na=False)]

    # 교육지원청 보정
    col_edu = [c for c in df.columns if '교육' in str(c) and '청' in str(c)][0]
    df = df.rename(columns={col_edu: '교육지원청_원본'})
    base = df[~df['학교급'].isin(['고등학교','고등기술학교','방송통신고등학교'])]
    real = base[~base['교육지원청_원본'].str.contains(
        r'시$|도$|특별자치시$|특별자치도$', na=False)]
    mp = real.groupby(['시도','행정구'])['교육지원청_원본']\
        .agg(lambda x: x.mode().iloc[0]).reset_index()
    mapping = dict(zip(zip(mp['시도'], mp['행정구']), mp.iloc[:,2]))

    def fix_edu(row):
        if row['학교급'] in ['고등학교','고등기술학교','방송통신고등학교']:
            return mapping.get((row['시도'], row['행정구']), row['교육지원청_원본'])
        if re.search(r'시$|도$|특별자치시$|특별자치도$', str(row['교육지원청_원본'])):
            return mapping.get((row['시도'], row['행정구']), row['교육지원청_원본'])
        return row['교육지원청_원본']

    df['교육지원청'] = df.apply(fix_edu, axis=1)
    df['학교급_분류'] = df['학교급'].map(LEVEL_MAP).fillna('기타')

    # 학교수 집계 분기
    has_school_col = '학교수' in df.columns
    agg_col  = '학교수' if has_school_col else '학교명'
    agg_func = 'sum'   if has_school_col else 'count'

    agg = df.groupby(['시도','학교급_분류']).agg(
        schools=(agg_col, agg_func),
        students=('학생수_총계_계', 'sum'),
        male=('학생수_총계_남', 'sum'),
        female=('학생수_총계_여', 'sum'),
    ).reset_index()

    records = [
        {'sido': r['시도'], 'level': r['학교급_분류'],
         'schools': int(r['schools']), 'students': int(r['students']),
         'male': int(r['male']), 'female': int(r['female'])}
        for _, r in agg.iterrows() if r['시도'] in SIDO_ORDER
    ]
    return actual_year, records
```
